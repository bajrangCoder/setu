# Setu

> **Note:** This project is currently a **Work in Progress (WIP)**. Features and APIs are subject to change.

**Setu** is a modern, minimal, and high-performance API testing application built with Rust and [GPUI](https://gpui.rs). Designed to be blazing fast and aesthetically pleasing, Setu aims to provide a seamless developer experience for testing and debugging HTTP APIs.

## Getting Started

Ensure you have [Rust](https://www.rust-lang.org/tools/install) installed on your system.

1. **Clone the repository:**

   ```bash
   git clone https://github.com/bajrangCoder/setu.git
   cd setu
   ```

2. **Run the application:**
   ```bash
   cargo run
   ```
